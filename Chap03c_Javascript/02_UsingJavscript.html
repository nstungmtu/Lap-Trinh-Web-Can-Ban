Sử dụng Javascript

Cách sử dụng Javascript

Sử dụng Javascript bên trong cặp thẻ <script></script>. Thẻ <script> có thể được đặt ở phần <head> hoặc bất cứ đâu bên trong thẻ <body>

Sử dụng tập tin Javascript bên ngoài. Để liên kết với tập tin Javascript bên ngoài chúng ta sử dụng khai báo <script src="ten_tap_tin.js"></script>

Kiểm tra kết quả đầu ra của Javascript

Sử dụng document.write()

Lưu ý: sử dụng document.write() sau khi trang web được tải sẽ ghi đè toàn bộ nội dung trang web.


function writeDocument(){
    document.write("oops!");
}

Sử dụng hàm innerHTML()

Hàm innerHTML() sử dụng trên DOM Element (phần tử Javascript tham chiếu đến phần tử trong tài liệu HTML), do đó, ta cần tạo tham chiếu phần tử HTML trước. Để tạo tham chiếu, ta có thể sử dụng hàm document.getElementById(id), tham số id là thuộc tính id của phần tử HTML.


function writeHMTL(){
    document.getElementById("vd2").innerHTML = Date.now();
}
Nhấn nút kế bên để thay đổi nội dung

Sử dụng hàm window.alert()

Hàm window.alert() bật một thông báo trên cửa sổ trình duyệt.


function writeAlert(){
    window.alert(Date.now());
}

Sử dụng hàm console.log()

Hàm console.log() ghi kết quả ra cửa sổ dòng lệnh của trình duyệt.


function writeConsole(){
    window.alert(Date.now());
}

Sử dụng hàm window.print()

Hàm window.print() mở chức năng in toàn bộ trang web của trình duyệt.


function writePrint(){
    window.print();
}
