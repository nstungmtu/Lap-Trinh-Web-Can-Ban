CSS = Cascading Style Sheets

CSS được dùng để định dạng cho các phần tử trong tài liệu HTML

Từ "Cascading" trong CSS có nghĩa nếu một style được áp dụng cho phần tử thì cũng mặc định được áp dụng cho các phần từ con của nó.

Ví dụ: nếu bạn đặt thuộc tính {color:blue} cho phần tử body, tất cả chữ hiển thị trên trang web đều sẽ mặc định có màu xanh

Ví dụ

Các phần tử bên trong thẻ div mặc định thừa kế thuộc tính color từ thẻ div nếu chúng ta không khai báo lại thuộc tính color cho các phần tử này

Đoạn văn bản thừa kế màu sắc từ cha của nó. Đoạn này định nghĩa lại thuộc tính màu sắc cho riêng nó

03 cách sử dụng CSS

Cách 1: Sử dụng inline-style. Khai báo css trong thuộc tính style của phần tử

Cách 2: Sử dụng internal-style. Khai báo css trong phần tử <style> ở phần <head> của tài liệu HTML

Cách 3: Sử dụng <link> để liên kết với tập tin css bên ngoài tài liệu HTML

Ví dụ

Tôi được định dạng bằng inline-style
Tôi được định dạng bằng internal-style
Tôi được định dạng bằng external-style
